**SLDS Select** — a native `<select>` styled to match SLDS inputs.

```jsx
<Select label="Stage" options={[
  { label: "Qualification", value: "q" },
  { label: "Proposal", value: "p" },
]} />
```

Props: `label`, `options` (`{label,value}` or strings), `value`/`defaultValue`, `onChange`, `required`, `disabled`, `error`.
