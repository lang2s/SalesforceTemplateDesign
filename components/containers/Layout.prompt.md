**SLDS Layout / Layout Item** — a 12-column responsive flex grid.

```jsx
<Layout gutters horizontalAlign="spread">
  <LayoutItem size={8}>Main</LayoutItem>
  <LayoutItem size={4}>Sidebar</LayoutItem>
</Layout>
```

`Layout`: `gutters`, `wrap`, `horizontalAlign` (`start`·`center`·`end`·`spread`·`space`), `verticalAlign`. `LayoutItem`: `size` (1–12), `flexibility`.
